## [0.3.0] - 2025-12-01

### Added

- Initial release of zeteo CLI
- OTEL log exploration capabilities
- Gemini AI integration
- Multiple backend support (Elasticsearch, Kibana, OpenObserve)
- Terminal user interface (TUI) for interactive log exploration
- Session management with Redis persistence
- MCP (Model Context Protocol) client integration
- Pre-commit hooks for code quality
- CI/CD pipelines with GitHub Actions
- Automated release workflow with version bumping
